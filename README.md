# Happy Birthday 🎂

An interactive digital birthday gift,  an animated site with a memory board, a virtual envelope card, and a custom soundtrack, built as a personal gift project.

## Features

- Animated envelope card reveal
- Interactive memory board component
- Background music
- Smooth animations powered by Framer Motion
- Data persistence via Supabase

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Supabase](https://supabase.com/) for backend/data storage

## Running Locally

```bash
git clone https://github.com/Keijoo/Happy-Birthday.git
cd Happy-Birthday
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Why I Built This

A one-off personal project to make a birthday gift more memorable than a card, a good excuse to get hands-on with Next.js, Framer Motion, and Supabase in a real (if small) full-stack app.
